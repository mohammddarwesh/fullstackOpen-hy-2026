# Full Stack Open 2026

This repository contains my solutions, notes, and exercises for the Full Stack Open 2026 course by the University of Helsinki.

---

## 📚 Course Overview

The course focuses on modern web development, covering:

- React
- JavaScript (ES6+)
- Node.js & Express
- REST APIs
- Single Page Applications (SPA)
- Full-stack development concepts

---

## 📁 Project Structure

Each part of the course is organized in its own directory:
part0/
part1/
part2/
part3/
...

Each folder contains solutions and related files for that part.
